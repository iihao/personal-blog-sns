### 4. 后端评论 API 修复

**问题:** 数据库使用 `status` 字段而非 `approved`

**解决:**
```javascript
// 支持多种查询方式
GET /api/comments?post_id=1&approved=true
GET /api/comments?post_id=1
GET /api/comments/post/1  (兼容旧版)
```

**状态映射:**
- `approved=true` → status IN ('approved', 'pending')
- `approved=false` → status = 'pending'
