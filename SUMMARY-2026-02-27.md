```javascript
// 5 个 Store 模块

usePostStore    - 文章管理 (CRUD + 分页 + 搜索)
useCommentStore - 评论管理
useMediaStore   - 媒体管理
useAuthStore    - 用户认证
```

**特性:**
- 统一 API 请求处理
- 错误处理机制
- 加载状态管理
- 本地存储 Token

**实现方式:**

- Composable 模式复用
- 本地存储用户偏好
- 自动检测系统主题

**覆盖范围:**
- 背景/文字/边框色彩
- 卡片/按钮/表单
- 代码块/引用块
- 滚动条/选中文本

### 3. Markdown 渲染

**支持语法:**
- 标题 (h1-h6)
- 粗体/斜体/删除线
- 代码块 (带语言标识)
- 行内代码
- 链接/图片
- 引用块
- 无序/有序列表
- 水平线
- 表格

**扩展功能:**
