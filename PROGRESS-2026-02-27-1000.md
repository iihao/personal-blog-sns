## 🎨 深色模式配色方案

### 主色调

| 元素 | 浅色模式 | 深色模式 |
|------|----------|----------|
| 背景主色 | `#ffffff` | `#0f172a` |
| 背景次色 | `#f9fafb` | `#1e293b` |
| 文字主色 | `#111827` | `#f1f5f9` |
| 文字次色 | `#6b7280` | `#94a3b8` |
| 强调色 | `#667eea` | `#60a5fa` |
| 边框色 | `#e5e7eb` | `#334155` |

### 渐变主题

**浅色模式:**
```css
background: linear-gradient(135deg, #f5f7fa 0%, #c3cfe2 100%);
```

**深色模式:**
```css
background: linear-gradient(135deg, #0f172a 0%, #1e293b 100%);
```

**标题渐变:**
- 浅色：`#667eea → #764ba2` (紫色系)
- 深色：`#60a5fa → #a78bfa` (蓝色系)

### 主题切换流程

```
1. 用户首次访问
   └─→ 检测系统偏好
       └─→ 应用对应主题

2. 用户手动切换

       └─→ 保存到 localStorage
```
