## 🎨 设计规范

### 配色系统
```css
/* 主色调 - 紫色渐变 */
--accent-primary: #7c3aed;
--accent-secondary: #3b82f6;

/* 中性色 */
--text-primary: #171717;
--text-secondary: #525252;
--text-tertiary: #a3a3a3;

/* 背景色 */
--bg-primary: #ffffff;
--bg-secondary: #fafafa;
--bg-tertiary: #f5f5f5;
```

### 圆角规范
| 级别 | 值 | 应用 |
|------|-----|------|
| 小 | `8px` | 按钮内部、小标签 |
| 中 | `10-12px` | 表单输入、工具栏 |
| 大 | `16-20px` | 卡片、容器 |

### 阴影规范
| 级别 | 值 | 应用 |
|------|-----|------|
| 默认 | `0 2px 8px rgba(0,0,0,0.08)` | 卡片 |
| 悬停 | `0 4px 16px rgba(0,0,0,0.1)` | hover |
| 发光 | `0 0 20px rgba(124,58,237,0.3)` | 主按钮 |

### 交互反馈
- 按钮 hover: `translateY(-2px)`
- 按钮 active: `translateY(0)`
- 卡片 hover: `translateX(4px)` / `translateY(-4px)`
- 输入框 focus: 边框 + 光晕
